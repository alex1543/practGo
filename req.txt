Сайт на Go (Colang) гарантированно работает при установленных программах и компонентах под Windows 11:
1) Go версии: go1.20.2 windows/amd64 в каталог: c:\Go\bin (из файла: go1.20.2.windows-amd64.msi)
1) git версии: 2.40.0.windows.1 (из файла: Git-2.40.0-64-bit.exe)
2) Go SQL Driver с помощью команды: go get -u github.com/go-sql-driver/mysql
